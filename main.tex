\section{Author Contributions}

The authors would like to acknowledge the work done by each of the author of this paper.

Lorenzo Gonzales worked on Sections \ref{kernel}, \ref{processor}, and \ref{memory},
which covers the topics of Kernel Management, Processor Management, and Memory Management
respectively. Furthermore, Gonzales contributed to Section \ref{overview}, specifically
Section \ref{netbsd} which introduces the NetBSD operating system. Lastly, Gonzales
wrote Section \ref{conclusion}, the last section of the paper summarizing the implementations
of FreeBSD and NetBSD, the major strengths and drawbacks of such implementations,
and on which workloads and use cases each operating system excels at.

Christian Brillos worked on Sections \ref{file-and-disk}, \ref{resource}, and \ref{concurrency},
which covers the topics of File and Disk Management, Resource Management, and Concurrency
Management respectively. Lastly, Brillos contributed to Section \ref{overview}, specifically
Section \ref{freebsd} which introduces the FreeBSD operating system.

Lastly, each author of this paper added and compiled their own reference entries.
